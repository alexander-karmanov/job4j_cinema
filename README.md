Сервис - Кинотеатр.

О проекте

Описание
Данное приложение - часть сервиса по онлайн-бронированию билетов в кинотеатр.
Войдя на сайт, пользователь может посмотреть актуальные сеансы, выбрать место и время сеанса. Для покупки билета пользователь должен войти на сайт под своим логином и паролем или пройти регистрацию.

Технологии:
    Java 18 - версия языка программирования Java
    Spring boot - фреймворк для разработки приложений на языке Java
    Liquibase - инструмент для управления версиями базы данных и автоматизации процессов миграции схемы данных
    JUnit - фреймворк для модульного тестирования ПО
    Mockito - библиотека для создания фиктивных объектов (моков) в тестах
    Maven - это инструмент для автоматической сборки проектов
    Tomcat - контейнер сервлетов с открытым исходным кодом,
    Thymeleaf -  шаблонизатор Java на стороне сервера
    PostgreSQL - объектно-реляционная СУБД
    Sql2o - библиотека для работы с базами данных
    H2 - СУБД для тестирования

Запуск проекта:
    Скопируйте репозиторий
    Создайте базу данных в PostgreSQL
    Настройте доступ к базе данных в файле application.properties
    Соберите и запустите проект
    Для доступа перейдите по адресу: http://localhost:8080/index
